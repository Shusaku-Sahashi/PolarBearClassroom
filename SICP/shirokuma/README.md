## 環境

* mac
* homeblewインストール済の前提で、以下を実行
  ```
  # racketインストール
  $ brew cask install racket

  # SICP学習用？のパッケージインストール
  $ raco pkg install sicp
  
  # 上記が成功するとdrracketを起動できる
  $ drracket
  ```
